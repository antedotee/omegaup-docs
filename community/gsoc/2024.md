# Google Summer of Code 2024

Projects completed during GSoC 2024.

## Projects

### Migrate Problem Creator to Vue.js + TypeScript
Migrated the Problem Creator to Vue.js + TypeScript for integration into omegaUp.com.

**Result**: Problem Creator can now be used directly on omegaUp.com without external website.

### Public Courses on GitHub
Enabled GitHub-based management of public course content with community contributions.

**Result**: Course content can be improved through pull requests.
