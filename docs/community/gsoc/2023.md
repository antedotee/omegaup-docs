# GSoC 2023

Projects completed during GSoC 2023.

## Projects

### Accounts for Children
Introduced restricted accounts for children under 13, allowing safe access to curated content.

**Result**: Children can safely use omegaUp with teacher assistance.

### Cypress Migration
Migrated Selenium tests to Cypress framework for better reliability and maintainability.

**Result**: Faster, more reliable E2E tests.
