# Google Summer of Code 2025

omegaUp is participating in Google Summer of Code 2025! We're looking for motivated students to contribute to our open-source educational platform.

## Project Ideas

### AI Teaching Assistant
Create a bot that can answer clarifications and perform code reviews proactively and upon request.

**Skills**: Python, PHP, MySQL, LLM Prompt Engineering, REST APIs  
**Size**: 350 hours  
**Level**: Advanced

### Generate Problem Editorials Using AI
Use generative AI to create and validate editorials for thousands of public problems.

**Skills**: Python, LLM Prompt Engineering, PHP, Vue.js  
**Size**: 350 hours  
**Level**: Advanced

### Public Courses on GitHub
Manage public course content through GitHub, allowing community contributions via pull requests.

**Skills**: git/github, Python, Continuous Integration, REST APIs  
**Size**: 175 hours  
**Level**: Medium to Advanced

### Cronjob Optimization
Optimize cronjobs for efficiency, increase test coverage, and improve debugability.

**Skills**: Python, MySQL, PHP, Unit and Integration Testing  
**Size**: 350 hours  
**Level**: High

### Integrate Problem Creator
Fully integrate the Problem Creator visual editor with omegaUp's create/edit problem workflows.

**Skills**: Vue.js, TypeScript, PHP  
**Size**: 350 hours  
**Level**: Medium

### Code Coverage for E2E Tests
Measure Cypress test coverage in Codecov to enforce minimum coverage levels.

**Skills**: Integration testing, TypeScript, PHP  
**Size**: 90 hours  
**Level**: Medium

## Application Process

### Phase 1: Complete Test
1. Create account at [omegaUp.com](https://omegaup.com)
2. Join [GSoC 2025 Test](https://omegaup.com/contest/gsoc2025)
3. Solve at least 2 of 3 problems

### Phase 2: Familiarize with Codebase
1. Set up [development environment](../../getting-started/development-setup.md)
2. Find and fix a bug from [Good first issues](https://github.com/omegaup/omegaup/labels/Good%20first%20issue)
3. Submit PR and get it merged

### Phase 3: Write Proposal
1. Craft design document using [proposal template](https://docs.google.com/document/d/1_FKfpc2M3VLDVYqvT8ZgsgwIJ3zaZnyUVmSm-H3h6UQ/edit)
2. Send draft for feedback via [form](https://forms.gle/TbbscnWA5B2ZWfJq7)
3. Submit final proposal to Google

### Phase 4: Interview
Selected candidates will be interviewed with behavioral and technical questions.

## FAQ

**Do I need to speak Spanish?**  
No, English is fine. We're inclusive to all languages.

**How many spots are available?**  
We're requesting 2 students, but final numbers will be confirmed in mid-May.

**How are students selected?**  
Based on skill level (PRs or experience), work plan quality, and cultural fit.
