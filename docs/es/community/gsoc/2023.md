Proyectos completados durante Google Summer of Code 2023.

---

## Proyectos completados

### Cuentas para niños (cumplimiento de COPPA)

**Descripción**:
Se introdujeron cuentas restringidas para niños menores de 13 años, lo que permite el acceso seguro a contenido seleccionado y al mismo tiempo se mantiene el cumplimiento de la COPPA (Ley de protección de la privacidad infantil en línea).

**Logros clave**:

- Verificación de edad implementada durante el registro.
- Se creó un tipo de cuenta restringida con recopilación de datos limitada.
- Flujo de trabajo de consentimiento de padres/maestros construido
- Diseñé el acceso a contenido curado para usuarios menores de edad.
- Cumplimiento garantizado de las normas de privacidad.

**Implementación Técnica**:

- Actualizaciones del esquema de base de datos para tipos de cuentas.
- Nuevo flujo de registro con verificación de edad.
- Sistema de gestión de consentimiento
- Filtrado de contenido para cuentas restringidas.

**Impacto**:
Los niños ahora pueden usar omegaUp de forma segura con la ayuda de un maestro, ampliando el alcance de la plataforma a los estudiantes más jóvenes mientras se mantiene el cumplimiento legal.

### Migración de cipreses

**Descripción**:
Migré el conjunto de pruebas de extremo a extremo existente basado en Selenium al marco Cypress para mejorar la confiabilidad, la mantenibilidad y la experiencia del desarrollador.

**Logros clave**:

- Migré más de 100 pruebas de Selenium a Cypress.
- Confiabilidad de las pruebas mejorada (pruebas menos fiables)
- Reducción del tiempo de ejecución de pruebas en un 40%.
- Se agregaron capacidades de prueba de regresión visual.
- Creé documentación de prueba completa.

**Pila técnica**:

- Ciprés 12.x
- TypeScript para código de prueba
- Acciones de GitHub para la integración de CI
- Percy para pruebas visuales (opcional)

**Beneficios**:

| Aspecto | Antes (selenio) | Después (Ciprés) |
|--------|-------------------|-----------------|
| Fiabilidad de la prueba | ~80% | ~98% |
| Tiempo de ejecución | ~45 minutos | ~27 minutos |
| Experiencia de depuración | Limitado | Excelente |
| Mantenimiento | Alto esfuerzo | Bajo esfuerzo |

**Impacto**:
El equipo de desarrollo ahora cuenta con pruebas E2E más rápidas y confiables, lo que genera ciclos de retroalimentación más rápidos e implementaciones más confiables.
