# Proceso de reclutamiento

¡omegaUp da la bienvenida a nuevos voluntarios para unirse a nuestro equipo de ingeniería! Este documento describe el proceso de capacitación para nuevos contribuyentes.

## Proceso de formación

El proceso de formación está abierto a cualquier persona interesada en contribuir. ¡No se necesita ninguna petición especial!

### Recursos

1. **Documentación**: [omegaUp Wiki](https://github.com/omegaup/omegaup/wiki)
2. **Tutorial en vídeo**: [Configuración del entorno de desarrollo](https://www.youtube.com/watch?v=H1PG4Dvje88)
3. **Discord**: [#dev_training canal](https://discord.com/invite/K3JFd9d3wk) para ayuda técnica
4. **Buenos primeros números**: [Números seleccionados para principiantes](https://github.com/omegaup/omegaup/issues?q=is%3Aissue+is%3Aopen+label%3A%22Good+first+issue%22)
5. **Buenos segundos números**: [Desafíos del siguiente nivel](https://github.com/omegaup/omegaup/issues?q=is%3Aissue+is%3Aopen+label%3A%22Good+second+issue%22)

## Requisitos para unirse al equipo

Para ser invitado formalmente al equipo de ingeniería:

1. ✅ Leer toda la documentación
2. ✅ Resuelva 5 problemas de la lista "Buenos primeros problemas"
3. ✅ Resuelva 5 problemas de la lista "Buenos segundos problemas"

Una vez completado, serás:
- Invitado formalmente a unirse al equipo de ingeniería.
- Se ofreció una cuenta de correo electrónico @omegaup.com (si se desea)

## Excepciones

Algunas excepciones a este proceso:
- Estudiantes en prácticas con contrato.
- Antiguos voluntarios con antecedentes de contribución reconocidos.
