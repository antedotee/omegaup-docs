# Concursos

omegaUp proporciona capacidades integrales de gestión de concursos para organizar concursos de programación.

## Tipos de concurso

- **Concursos Públicos**: Abierto a todos los usuarios
- **Concursos privados**: restringido a participantes invitados
- **Concursos virtuales**: comience en cualquier momento dentro de una ventana
- **Estilo USACO**: Cronómetro individual por participante

## Características del concurso

- **Gestión de problemas**: agregar/eliminar problemas, establecer valores de punto
- **Gestión de participantes**: invitar a usuarios, gestionar equipos
- **Marcador**: clasificaciones en tiempo real con visibilidad configurable
- **Aclaraciones**: Sistema de preguntas y respuestas durante los concursos
- **Sistema de penalización**: Cálculo de penalización configurable
- **Descenso de puntuación**: descenso de puntuación opcional basado en el tiempo
