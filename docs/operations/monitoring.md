# Monitoring

omegaUp uses various monitoring tools to track system health and performance.

## Monitoring Tools

- **Application Monitoring**: Track application performance
- **Database Monitoring**: MySQL performance metrics
- **Server Monitoring**: System resources and health
- **Error Tracking**: Error logging and alerting

## Key Metrics

- **Response Times**: API endpoint performance
- **Error Rates**: Failed requests and exceptions
- **Database Performance**: Query times and connections
- **Submission Queue**: Grader queue length
- **Runner Status**: Runner availability and health
