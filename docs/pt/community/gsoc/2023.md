Projetos concluídos durante o Google Summer of Code 2023.

---

## Projetos Concluídos

### Contas para crianças (conformidade com a COPPA)

**Descrição**:
Introduziu contas restritas para crianças menores de 13 anos, permitindo acesso seguro ao conteúdo selecionado, mantendo a conformidade com a COPPA (Lei de Proteção à Privacidade Online das Crianças).

**Principais conquistas**:

- Implementada verificação de idade durante o registro
- Criado tipo de conta restrita com coleta de dados limitada
- Fluxo de trabalho de consentimento dos pais/professores criado
- Acesso a conteúdo com curadoria projetada para usuários menores de idade
- Garantiu a conformidade com os regulamentos de privacidade

**Implementação Técnica**:

- Atualizações de esquema de banco de dados para tipos de conta
- Novo fluxo de cadastro com verificação de idade
- Sistema de gerenciamento de consentimento
- Filtragem de conteúdo para contas restritas

**Impacto**:
As crianças agora podem usar o omegaUp com segurança com a assistência do professor, expandindo o alcance da plataforma para alunos mais jovens e mantendo a conformidade legal.

### Migração Cipreste

**Descrição**:
Migrou o conjunto de testes ponta a ponta existente baseado em Selenium para a estrutura Cypress para melhorar a confiabilidade, a capacidade de manutenção e a experiência do desenvolvedor.

**Principais conquistas**:

- Migração de mais de 100 testes Selenium para Cypress
- Maior confiabilidade do teste (redução de testes instáveis)
- Tempo de execução de teste reduzido em 40%
- Adicionados recursos de teste de regressão visual
- Criação de documentação abrangente de testes

**Pilha Técnica**:

- Cipreste 12.x
- TypeScript para código de teste
- Ações GitHub para integração de CI
- Percy para testes visuais (opcional)

**Benefícios**:

| Aspecto | Antes (Selênio) | Depois (Cipreste) |
|--------|-------------------|-----------------|
| Confiabilidade do teste | ~80% | ~98% |
| Tempo de Execução | ~45 minutos | ~27 minutos |
| Experiência de depuração | Limitado | Excelente |
| Manutenção | Alto esforço | Baixo esforço |

**Impacto**:
A equipe de desenvolvimento agora conta com testes E2E mais rápidos e confiáveis, levando a ciclos de feedback mais rápidos e implantações mais confiáveis.
