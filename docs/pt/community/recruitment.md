# Processo de recrutamento

omegaUp dá as boas-vindas a novos voluntários para se juntarem à nossa equipe de engenharia! Este documento descreve o processo de treinamento para novos colaboradores.

## Processo de treinamento

O processo de treinamento está aberto a qualquer pessoa interessada em contribuir. Não é necessário nenhum pedido especial!

### Recursos

1. **Documentação**: [omegaUp Wiki](https://github.com/omegaup/omegaup/wiki)
2. **Tutorial em vídeo**: [Configuração do ambiente de desenvolvimento](https://www.youtube.com/watch?v=H1PG4Dvje88)
3. **Discord**: [#dev_training channel](https://discord.com/invite/K3JFd9d3wk) para ajuda técnica
4. **Boas primeiras questões**: [Problemas selecionados para iniciantes](https://github.com/omegaup/omegaup/issues?q=is%3Aissue+is%3Aopen+label%3A%22Good+first+issue%22)
5. **Boas segundas questões**: [Desafios de próximo nível](https://github.com/omegaup/omegaup/issues?q=is%3Aissue+is%3Aopen+label%3A%22Good+second+issue%22)

## Requisitos para ingressar na equipe

Para ser formalmente convidado para a equipe de engenharia:

1. ✅ Leia toda a documentação
2. ✅ Resolva 5 problemas da lista de "Primeiros problemas bons"
3. ✅ Resolva 5 problemas da lista de "Segundos problemas bons"

Depois de concluído, você será:
- Convidado formalmente para integrar a equipe de engenharia
- Ofereceu uma conta de e-mail @omegaup.com (se desejar)

## Exceções

Algumas exceções a este processo:
- Alunos estagiários com contrato
- Ex-voluntários com registros de contribuição reconhecidos
